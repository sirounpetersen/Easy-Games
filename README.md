## Installation

Use the package manager pip3 to install the following dependencies:
  *Flask

```bash
sudo pip3 install flask
```

## Usage

```python
from flask import Flask, request, render_template, flash
```

## To Run
```bash
export FLASK_APP=main.py
flask run --host=0.0.0.0
```
